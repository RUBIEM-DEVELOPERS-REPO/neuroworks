# Skill: Event run-sheet

## Goal

On the day, anyone holding the run-sheet knows exactly what happens, when, and who's responsible for it — so the event runs without the organiser being asked 200 questions.

## Structure

```
# Run-sheet — <Event> · <date> · <venue>
Organiser: <name/phone> · On-site lead: <name/phone> · Doors: <time>

## Key contacts
| Role | Name | Phone |
|---|---|---|
| Venue | | |
| AV / tech | | |
| Catering | | |
| Speakers liaison | | |

## Timeline (minute-by-minute)
| Time | What | Owner | Notes / cue |
|---|---|---|---|
| 07:00 | Pack-in, AV setup | Ops | sound check by 08:00 |
| 08:30 | Registration opens | Front desk | badges A–M / N–Z |
| 09:00 | Welcome | MC | mic 1, slides cued |
| 09:15 | Keynote | Speaker 1 | 30 min + 5 Q&A |
| 10:00 | Break | Catering | coffee in foyer |
| … | | | |
| 16:30 | Close + pack-out | Ops | venue cleared by 18:00 |

## Logistics
- Setup/layout · AV requirements · catering counts + dietary · signage · Wi-Fi · accessibility · parking

## Roles on the day
| Person | Owns |
|---|---|

## Contingencies
- Speaker no-show → <backup / reorder>
- AV failure → <fallback> · Over-run → <where to cut> · Weather (if outdoor) → <plan B> · Medical → <first aid + numbers>
```

## Rules

- **Every line has a time AND an owner.** A cue with no owner is a cue that gets missed.
- **Minute-by-minute through the busy parts** (pack-in, registration, transitions); looser through steady-state.
- **Build in buffers** — transitions and Q&A always over-run.
- **Contingencies for the predictable failures** (speaker, AV, over-run, weather, medical).
- **One on-site lead** as the single decision-maker; everyone else escalates to them.
- **Pack-in and pack-out** are part of the plan, not afterthoughts (venue hand-back times are real deadlines).

## Pitfalls

- A schedule with no owners — the organiser becomes the bottleneck.
- No buffer time — one 10-min slip cascades all day.
- Forgetting pack-out / venue hand-back — overage fees.
- Contacts not on the sheet — chasing a phone number mid-crisis.
