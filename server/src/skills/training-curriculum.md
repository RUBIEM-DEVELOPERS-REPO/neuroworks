# Skill: Training curriculum

## Goal

A learner finishes able to DO something they couldn't before — because the curriculum starts from measurable objectives, sequences from foundational to advanced, and checks mastery along the way.

## Structure

```
# Curriculum — <Topic / Role> · <audience> · <total duration>
**Outcome:** by the end, learners can <observable capability>.
**Prerequisites:** <what they must already know/have>.

## Learning objectives (Bloom verbs — observable)
1. Learners will be able to <identify / apply / analyse / build> <X>.
2. …

## Module plan
| # | Module | Duration | Objective covered | Method | Assessment |
|---|---|---|---|---|---|
| 1 | Foundations | 45 min | LO1 | demo + discussion | quick check |
| 2 | Hands-on | 90 min | LO2-3 | exercise | graded task |
| 3 | … | | | | |

## Delivery
- Format (live / self-paced / blended) · cohort size · materials · tools needed

## Assessment & mastery
- Formative (during): <checks for understanding>
- Summative (end): <quiz / project / sign-off> — pass bar: <e.g. 80%>

## Reinforcement
- <post-training: job aid, refresher, manager check-in at 30 days>
```

## Rules

- **Objectives use observable verbs** (apply, build, diagnose) — never "understand" or "be aware of", which can't be assessed.
- **Sequence foundational → advanced**; each module builds on the last and maps to ≥1 objective. No orphan content.
- **More doing than telling.** Adults learn by practising — bias to exercises, not lecture.
- **Assess against the objectives**, not trivia — if LO1 is "configure X", the assessment is "configure X", not "list the steps".
- **Realistic durations with breaks**; attention fades after ~50 minutes.
- **Plan reinforcement** — training with no follow-up is forgotten in a week.

## Pitfalls

- "Understand/know" objectives that can't be measured.
- Content with no matching objective (cut it) or objectives with no content (gap).
- All lecture, no practice — knowledge that doesn't transfer.
- No mastery bar — "attended" mistaken for "can do".
