# Skill: Negotiation brief

## Goal

Walk into the negotiation knowing your target, your walk-away, what you'll trade, and what the other side likely wants — so you concede deliberately, not reactively.

## Structure

```
# Negotiation brief — <deal / counterparty>
**Objective:** <the outcome that = success in one line.>
**Our BATNA:** <best alternative if this deal dies — the real source of power.>
**Their likely BATNA:** <what they do if we walk — drives their flexibility.>

## Interests (not positions)
- Ours: <what we actually need underneath the ask>
- Theirs: <what they actually need — infer from their business>

## The variables (trade these, don't just split price)
| Variable | Our target | Acceptable | Walk-away | Value to them |
|---|---|---|---|---|
| Price | <x> | <range> | <floor/ceiling> | high |
| Term length | | | | |
| Payment terms | | | | |
| SLA / penalties | | | | |
| Exclusivity / volume | | | | |

## ZOPA
<Where our acceptable range overlaps theirs — if any. If none, change the variables or walk.>

## Plan
- **Open at:** <ambitious but defensible anchor + the why>
- **Concession ladder:** give <low-cost-to-us / high-value-to-them> first; each concession asks for something back.
- **Walk-away line:** <the point we leave — pre-committed so we don't cave in the room>
- **Traps to avoid:** <deadline pressure, splitting the difference reflexively, anchoring to their number>
```

## Rules

- **Know your BATNA before the room.** Your power is your alternative, not your argument.
- **Negotiate interests, trade variables** — never let it collapse to a single number. Bundle price with term, volume, SLA, payment timing.
- **Anchor first and high(-ish)** with a defensible rationale; the first number frames the range.
- **Every concession buys something back.** Unilateral giving trains the other side to keep pushing.
- **Pre-commit the walk-away.** Decide it cold, in advance, so emotion doesn't move it.

## Pitfalls

- Going in with a target but no walk-away — you'll accept a bad deal to avoid "failing".
- Arguing positions ("we want 20%") instead of interests ("we need predictable cost").
- Splitting the difference as a reflex — rewards their extreme anchor.
- Conceding on price first — the easiest thing to give and the hardest to get back.
