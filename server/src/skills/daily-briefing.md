# Skill: Daily briefing

## Goal

A 30-second read at the start of the day that answers three questions:
**what does today actually look like, what does the operator owe people from
yesterday, and what are the top 3 things to push on.** Not a to-do list —
a situational brief.

## Format

```
# <Operator name> — Briefing for <Day, YYYY-MM-DD>

## Today's shape
- **Meetings:** <count and a one-line shape — e.g. "3 meetings, all back-to-back 10-12">
- **First / last commitment:** <times>
- **Focus blocks:** <count of 60-minute+ uninterrupted blocks, or "none">

## Carryover from yesterday
- <Concrete open item, with where it sits and who's waiting>
- <...>

## Inbox — needs you
- **<sender / subject>** — <one-line summary of what they're asking and the deadline>
- <...>

## Top 3 today
1. <Outcome, not activity — "ship X" not "work on X">
2. <...>
3. <...>

## Watch-list *(optional)*
- <Thing that's not blocking but worth noticing — quietly slipping, person to check on, deadline creeping>
```

## Rules

- **The shape matters more than the list.** "3 meetings, all morning" tells the operator more than enumerating them.
- **Carryover is what was promised, not what was wished.** Only include items the operator explicitly said they'd do.
- **Inbox "needs you" excludes notifications, newsletters, autoreplies.** Include only mail with an implicit ask.
- **Top 3 = outcomes.** "Decide X with Sarah" beats "look at X". An outcome forces a decision.
- **No filler.** If there's no carryover, drop the section. A brief with empty sections feels mechanical.

## Length

150-300 words. A briefing the operator skims standing up.

## Tone

Direct, brief, declarative. No "I noticed", "you might want to consider".

## Pitfalls

- Reading like an agenda for someone else → write it for the operator, second person voice ok.
- Listing every email → bury the signal.
- Generic priorities ("be productive") → if you can't name a concrete outcome, drop the slot.
- Speculating about what others want → only include what's stated in the artefacts (calendar, inbox, carryover notes).
