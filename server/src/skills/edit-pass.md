# Skill: Edit pass

## Goal

Return a version of the draft that's tighter, clearer, and still sounds like the author wrote it. Not a rewrite — an edit.

## What to fix (in order)

1. **Logic and accuracy first.** A typo doesn't matter if a claim is wrong. Read once for sense before touching language.
2. **Structure.** Are paragraphs in the right order? Does the lead carry the point? Cut sections that add no information.
3. **Sentences.** Trim filler ("in order to" → "to"; "due to the fact that" → "because"). Replace passive with active where it sharpens responsibility.
4. **Words.** Concrete > abstract. Specific > generic. "12%" > "significant". Cut adverbs that lean on adjectives ("very important" → "important" or pick a stronger adjective).
5. **Mechanics.** Spelling, grammar, punctuation. Do this LAST — earlier passes shuffle text and create new typos.

## Output shape

Default: the edited draft, ready to ship. No explanations, no "Here's the revised version".

When the customer asks for a **review** instead of a clean edit:
```
**Top-line:** <one sentence summary of how the draft lands>

**Strengths:**
- <thing the author got right — keep doing this>

**Suggested changes:**
- <change 1 + one-line reason>
- <change 2 + one-line reason>

**Optional polish:**
- <small things — author's call>
```

## Rules

- **Preserve voice.** If the author writes punchy sentences, don't make them long. If they're formal, don't go colloquial.
- **Show, don't tell, what changed.** When the customer needs to see diffs, use ~~strikethrough~~ for cuts and **bold** for new text inline. Otherwise just hand them the clean version.
- **Cut more than you add.** Most edits should shrink word count by 10-30%.
- **Don't add facts.** If the draft is missing evidence, flag it ("source needed for the 12% claim") — don't invent the source.
- **Question the title and the opening line.** They do the most work. If they're weak, the rest of the edit barely matters.

## Pitfalls

- "Improving" the author's voice into your own → the author rejects the whole edit.
- Fixing only mechanics → the structural problems ship.
- Adding throat-clearing the author cut on their own draft ("It is important to note that…").
- Leaving "TBD" in a place the author meant to fill — flag it explicitly.
