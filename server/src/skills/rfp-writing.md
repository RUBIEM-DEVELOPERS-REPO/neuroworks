# Skill: RFQ / RFP draft

## Goal

Vendors submit comparable, complete bids you can score objectively — because the document told them exactly what you need, how you'll judge it, and how to respond.

## Structure

```
# Request for <Quotation | Proposal> — <what you're buying>
Issued by: <Org> · Ref: <RFP-####> · Issued: <date> · Responses due: <date/time + timezone>

## 1. Background & objective
<Who you are, the problem, the outcome you want from this purchase.>

## 2. Scope of work / specification
<Exactly what the vendor must deliver — quantities, specs, service levels, integrations.>

## 3. Requirements
### Mandatory (pass/fail)
- M1 <must-have — disqualifying if absent>
### Scored (weighted)
- S1 <capability> — weight <%>

## 4. Evaluation criteria
| Criterion | Weight | How scored |
|---|---|---|
| Technical fit | 40% | <rubric> |
| Price (TCO) | 30% | <lowest = full marks…> |
| Delivery / SLA | 15% | |
| Vendor stability / references | 15% | |

## 5. Commercials
<Pricing format required (unit + TCO), currency, payment terms, validity period.>

## 6. Submission instructions
<Format, channel, what to include, page limits, contact for questions, Q&A deadline.>

## 7. Timeline
| Milestone | Date |
|---|---|
| Questions due / Q&A published | |
| Responses due | |
| Shortlist / demos | |
| Award | |

## 8. Terms
<Confidentiality, right to reject any bid, no-obligation, conflict-of-interest declaration.>
```

## Rules

- **Separate mandatory (pass/fail) from scored (weighted)** — and publish the weights. Hidden criteria produce un-comparable, un-defensible bids.
- **Specify the response format and pricing format** so bids are apples-to-apples (unit price AND total cost of ownership).
- **State evaluation criteria up front** — fairness and a clean audit trail; required for public/regulated buyers.
- **Give a real Q&A window** and publish answers to all bidders.
- **Define the timeline and the contact** — single point of contact prevents back-channel lobbying.

## Pitfalls

- Vague scope → vendors quote different things → impossible to compare.
- Scoring criteria invented after bids arrive — unfair and challengeable.
- Asking for price without TCO terms — lowest sticker hides setup/lock-in costs.
- No mandatory/scored split — a missing dealbreaker slips through on a high total.
