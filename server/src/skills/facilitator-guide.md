# Skill: Workshop facilitator guide

## Goal

A facilitator (even one who didn't design it) can run the session confidently — they know the goal, the timing, what to say, what to ask, and how to handle the room.

## Structure

```
# Facilitator guide — <Workshop> · <duration> · <group size>
**Goal:** <what the group produces / decides / leaves able to do.>
**Outputs:** <tangible artefacts the session creates.>

## Before
- Room/virtual setup · materials · pre-reads sent · roles (facilitator, scribe, timekeeper)

## Run of show
| Time | Segment | Activity | Facilitator does / says | Output |
|---|---|---|---|---|
| 0:00 | Open | framing + agenda | "We're here to… by the end we'll have…" | shared goal |
| 0:10 | Diverge | brainstorm (silent → share) | prompt: "<question>"; timebox 8 min | idea set |
| 0:30 | Converge | dot-vote / cluster | "Pick the 3 that…" | shortlist |
| … | | | | |
| close | Wrap | decisions + owners | recap, assign next steps | action list |

## Facilitation notes
- Energy management: <when to break, switch modes>
- Prompts to unstick a quiet room / redirect a dominator
- Parking lot for off-topic items

## After
- Send notes + decisions + owners within 24h
```

## Rules

- **Timebox everything** and put the clock in the guide — workshops over-run by default.
- **Alternate diverge (generate) and converge (decide)** modes; don't brainstorm and judge at once.
- **Give the facilitator actual talk-tracks and prompts**, not just topic labels — that's the difference between a guide and an agenda.
- **Every segment produces a visible output** (list, vote, decision) — momentum comes from artefacts.
- **End with decisions + owners + next steps**, captured live, sent within a day.
- **Plan for the room**: a prompt to draw out the quiet, a move to handle the dominator, a parking lot for tangents.

## Pitfalls

- A bare agenda with no instructions — a stand-in facilitator is lost.
- All discussion, no structured activity — the loudest voice wins, energy dies.
- No timeboxes — segment one eats the whole session.
- No captured outputs — everyone leaves with a different memory of what was decided.
